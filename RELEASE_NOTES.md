### Double-click to preview
Double-click any file on a disc to open it straight in your system's default app — no manual extraction needed.

- **Pictures, video, music, documents, fonts** — jpg/png/tif/psd…, mp4/avi/mkv…, mp3/flac/wav/mid/voc…, pdf/txt/Office/epub…, ttf/otf.
- **Read-only previews** — the temp copy is write-protected, so edits in a viewer can't be mistaken for changes to the disc image.
- **Nested disc images open in Disc Xplorer** — double-click an `.iso`, `.chd`, `.nrg`, `.wbfs`, etc. stored *inside* a disc and browse it directly (self-contained formats; multi-file `.cue`/`.mds` sets still need a manual extract).
- Executables are deliberately not double-clickable — opening one would run it.
- Preview temp files are cleaned up automatically when the app closes (and on startup after a crash).

### Other
- Works from every filesystem view: ISO 9660, Joliet, HFS, UDF, FATX, KryoFlux/FAT floppies.
