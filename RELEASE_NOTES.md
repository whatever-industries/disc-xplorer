### Right-click to download
The browser's default context menu (Back / Refresh / Save as / Print) is gone. Right-clicking now shows a proper **⬇ Download** menu on:

- **Sidebar folders** — downloads the folder recursively
- **Sidebar filesystem roots** (ISO 9660, Joliet, HFS, …) — downloads that entire filesystem
- **File-list rows** — files and folders, same as their save button
- **Audio tracks** — saves the track in your configured format
