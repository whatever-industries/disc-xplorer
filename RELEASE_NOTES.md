### Latest Date Finder 📅
A new opt-in tool for dating a disc mastering. Enable **Latest Date Finder** in Settings and a 📅 button joins the toolbar whenever an image is open (tracks view included). One click reports:

- **PVD volume dates** — created and modified, with timezone; on multi-session discs the newest session's dates win
- **Newest file/folder date** on the disc, with the full path of the entry that carries it
- Scans the **entire disc** regardless of the folder you're browsing — on multi-session images (e.g. DiscJuggler `.cdi`) every session's data track is walked, and the newest entry is labeled with its track

### Fixes
- Multi-session `.cdi` images: session enumeration no longer stops at the first data track internally, so whole-disc operations see every session
