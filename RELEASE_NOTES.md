### Disc navigation and damage-map reliability
Disc Xplorer now handles a wider range of real-world mastering layouts without false red-X indicators, parse errors, or failures while opening nested folders.

- Correct CD-i Green Book descriptors, early prototype signatures, CD-i Ready pregaps, and Form 2 quality-control bytes
- Reliable multisession CUE, MDS/MDF, and skeleton navigation, including files whose extents point into an earlier data track
- Fixed CSO block decoding and reads that cross compressed-block boundaries
- Xbox XDVDFS empty-directory filler is no longer exposed as corrupt files
- Raw FAT12/FAT16 floppy images can be browsed and extracted directly
- Empty images and metadata-only TOC sidecars now show an explanatory message instead of being misidentified as ISO 9660
